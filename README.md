# 버튼 잡기 게임 (C# 코딩)

## 개요
- C# 프로그래밍 학습
- 핵심기능: 버튼 컨트롤 영역 안으로 마우스가 들어가면 버튼을 랜덤 위치로 이동, 버튼 클릭 시 점수 변경
- 화면구성: 버튼, 마우스, 확인 창
